# Security Policy

This repository contains mathematical source code and a manuscript; it does not provide a network service or process untrusted input in production.

For a security-sensitive issue involving the repository, its GitHub Actions workflow, or a dependency configuration, use GitHub's private vulnerability reporting feature when available. Otherwise, open an issue that requests private follow-up without including exploit details. For ordinary mathematical, Lean, build, or documentation issues, open a public GitHub issue.

Only the current `main` branch and the latest tagged release are supported.
